# 加入我们

如果你有项目开发的经历，或者想帮助我们测试出软件的`bug`，或者仅仅是想要获得义管会的奖励，这个页面会指示你参与项目发展。

## 获取一个账号

在[Gitee](https://gitee.com)注册一个账号即可

## 找到zvms的项目

打开zvms组织的[网址](https://gitee.com/zvms)
